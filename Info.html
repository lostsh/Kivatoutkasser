Hégerger une page html / PHP

Copiez votre page dans le repertoire /var/www/html/ puis votre page sera accessible.
Pour acceder a la page web, vous pouvez entrer dans le navigateur l'url du raspberry pi c'est a dire soit son ip du type 192.168.1.* ou encore si vous ne connaissez pas l'ip de ma pitite framboise vous pouvez aussi entrer son nom c'est a dire http://zthost

Une fois la page en place vous pouvez parametrer le SGBD et les bases pour qu'elles répondent a vos attententes, en utilsant l'interface phpmyadmin a laquelle vous pouvez vous connecter tout simplement a l'url http://zthost/phpmyadmin puis nom d'utilisateur et passwd comme dans le tableau en haut de page.
